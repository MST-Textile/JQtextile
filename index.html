JQtextile A textile interpreter for jQuery.

JQtextile is a textile interpreter plug-in for the jQuery javaScript framework.

Dependencies

JQuery 1.6.x.

Demonstration

Here you can see an example of JQtextile in action. The input here is taken from the contents of an element, but could just as easily be a string of text extracted from a JSON object.
